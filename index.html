CSS (Sass)

CSS is used to handle all aspects of the animation. Older browsers will simply fallback to a static version. This is perfectly normal and the way we design and develop for the modern web.

  • nestle.sass
  • tmp.sass (Not required)

Layers

A 3D rendering of the module. Helpful in understanding positioning and placement of the CSS z-index layers.

Performance

Performance of your website is very important to TMP. We take pride in optimizing your experience so that all user can enjoy it. The total size of this module is ~183KB. Great care has also been taken to produce well-balanced web animation, so as to not have it hinder the overall experience.

Photoshop Assets

  • Mobile (4.3MB)
  • Desktop (10.8MB)

Dependencies

Picturefill 3.0.2 is required to support the new HTML5 picture element in non-supporting browsers, like Internet Explorer 11.

A custom version of Modernizr is required to support animation in older browsers. CSS animation only runs when the "cssanimations" class is present in the opening html element.

Browser Support

Please see TMP Worldwide's Operating System and Browser support listing. TMP also attempts to support as much of WCAG 2.0 Level AA as is possible on every project.
